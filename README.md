Image Generation

About:
- Evolving images with genetic algorithms on the GPU

Using:
- OpenGL 4
- GLFW

Platforms:
- Windows
- Linux

Controls:
- WASD - Change current grid to render
- 1 - Set grid control to manual
- 2 - Set grid control to automatic
- F2 - Screenshot
- F5 - Save
- F9 - Load
- T - Print results
- Enter - View the current generation or overall picture
- Space - Toggle pause
